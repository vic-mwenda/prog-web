## About

The Coding Program School Portfolio is a collection of projects that were completed as part of a coding program or bootcamp. Each project demonstrates the skills and knowledge gained during the learning journey. The website serves as a showcase to potential employers or clients, displaying the ability to create dynamic and interactive web applications.

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

To view the portfolio locally or contribute to the project, follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/vic-mwenda/prog-web
```

2. Navigate to the project directory:
```bash
cd coding-program-portfolio
```


3. Open the `index.html` file in your web browser.

## Features

- Interactive and responsive design
- Project cards with project details and links to live demos
- Sections for different types of projects (e.g., web development, JavaScript applications, etc.)

## Contributing

If you want to contribute to this project and add your own projects to the portfolio, follow these steps:

1. Fork the repository.
2. Create a new branch:
```bash
git checkout -b add-my-project
```
3. Add your project files and update the `index.html` accordingly.
4. Commit your changes:
```bash
git commit -m "Add my project"
```
5. Push to the branch:
```bash
git push origin add-my-project
```
6. Create a pull request on GitHub.
